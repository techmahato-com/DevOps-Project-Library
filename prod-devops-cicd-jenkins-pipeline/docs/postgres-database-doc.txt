PostgreSQL Quick Reference
==========================

## Check Status
systemctl status postgresql
systemctl is-active postgresql


## Login as postgres superuser
sudo -u postgres psql


## Create User
CREATE USER sonaruser WITH PASSWORD 'sonaruser';


## Create Database
CREATE DATABASE sonarqubedatabase OWNER sonaruser;


## Grant All Privileges
GRANT ALL PRIVILEGES ON DATABASE sonarqubedatabase TO sonaruser;


## Check All Databases
\l

## Check All Users
\du

## Check Current Connections
\conninfo


## Login as sonaruser to sonarqubedatabase
psql -U sonaruser -d sonarqubedatabase -h localhost


## Exit psql
\q


## Restart / Stop / Start
systemctl restart postgresql
systemctl stop postgresql
systemctl start postgresql


## View PostgreSQL logs
journalctl -u postgresql -f
